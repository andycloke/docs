title: "Add request description"
---

You can add a description to your request to include helpful details or notes that make it easier to remember its purpose. Descriptions appear when hovering over the info icon (ℹ️) on both the request and home screens.

## **How to add a description to a request**

Follow these steps to add a description to your Data Fetcher request:

1. On the request screen, click the info icon (ℹ️) located next to the request name at the top of the screen.
2. Enter your description text in the input field that appears.
3. Click **Confirm** to save your description.
4. Click **Save** to finalize your request with the new description.
5. Once saved, you can view the description by hovering over the info icon on either the request screen or the home screen.

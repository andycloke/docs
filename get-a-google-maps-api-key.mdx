title: "Get a Google Maps API key"
mode: "wide"
---

When using the Google Maps no-code integration in Data Fetcher, you will need to add an API key. You create this API key in Google Cloud Console. The steps to do this are different for different endpoints. Use the instructions for the endpoint you are using:

| **Endpoints**                                                                                                      | **Step 1 (Billing & Enable API) Instructions**                                  | **Step 2 (Create API Key) Instructions**                                                          | **Pricing Details**                                                                   |
| ------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| Find a place from name/address/phone number<br /><br />Find details for a place id                                 | https://developers.google.com/maps/documentation/places/web-service/get-api-key | https://developers.google.com/maps/documentation/places/web-service/get-api-key#creating-api-keys | https://developers.google.com/maps/documentation/places/web-service/usage-and-billing |
| Calculate distance and time between start and end locations                                                        | https://developers.google.com/maps/documentation/distance-matrix/cloud-setup    | https://developers.google.com/maps/documentation/distance-matrix/get-api-key#creating-api-keys    | https://developers.google.com/maps/documentation/distance-matrix/usage-and-billing    |
| Geocode - find latitude/ longitude from address<br /><br />Reverse geocode - find address from latitude/ longitude | https://developers.google.com/maps/documentation/geocoding/get-api-key          | https://developers.google.com/maps/documentation/geocoding/get-api-key#creating-api-keys          | https://developers.google.com/maps/documentation/geocoding/usage-and-billing          |
